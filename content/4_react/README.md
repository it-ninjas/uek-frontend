# 4 React

* React ist 'nur' eine **view library** und kein komplettes SPA framework
* Aus diesem Grund wird React oft mit anderen Libraries kombiniert:
  * Routing
  * Backend API access
  * i18n
  * styling (bootstrap, ...)
  * State Management
